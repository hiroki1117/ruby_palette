# Devise Rails

Devise調査用

## 機能 (Features)

このアプリケーションは、以下の機能を提供します。

* **ユーザー認証機能**
  * 新規登録
  * ログイン
  * ログアウト

* **ユーザー一覧表示機能**
  * 登録されている全ユーザーのIDとメールアドレスを一覧で表示します。
  * このページはログインしているユーザーのみがアクセスできます。

## URL

* **トップページ**: `/`
* **新規登録**: `/register`
* **ログイン**: `/login`
* **ログアウト**: `/logout` (DELETEリクエスト)
* **ユーザー一覧**: `/users`
